# PF2e KeyRing

Модуль для Foundry VTT + Pathfinder 2e: трекер абстрактных ключей и квестовых предметов для партии.

GM ведёт каталог ключей, игроки видят только то, что уже есть у группы. В чате работают кнопки `@KeyTake` и `@KeyCheck`.

## Возможности

- Каталог ключей (GM): название, описание, GM-заметки, тексты успеха/провала проверки
- Опциональная связь с UUID предмета PF2e
- Состояние партии: кто носит ключ
- Кнопки в чате: `@KeyTake[id]{label}` и `@KeyCheck[id]{label}`
- Редактор (только GM): `Alt+K` или кнопка на панели токенов
- HUD статуса ключей (GM видит всё, игроки — только свои)

## Установка

В Foundry VTT укажите ссылку на манифест:

```text
https://raw.githubusercontent.com/AnkeKatomeky/pf2e-keyring/main/module.json
```

## Совместимость

- Foundry VTT v13+
- Pathfinder 2e v7+

## Разработка

Сборка не требуется. Точка входа: `scripts/main.js`.
